# 🎤 InterviewPrep AI

An AI-inspired interview preparation platform built using **Flutter** and **FastAPI** that helps users prepare for technical and HR interviews across multiple IT job roles.

---

## 📖 Overview

InterviewPrep AI allows users to select an IT job role and interview difficulty level to instantly generate technical and HR interview questions.

The application is designed to help students, fresh graduates, and job seekers practice interviews in an interactive and user-friendly environment.

---

## ✨ Features

* 💼 Supports 30+ IT Job Roles
* 📚 Easy, Medium, and Hard Difficulty Levels
* 🎯 Role-Based Technical Questions
* 👨‍💼 Random HR Interview Questions
* ⚡ FastAPI REST API Backend
* 📱 Flutter Frontend
* 🔄 Dynamic Question Generation
* 🎨 Modern Material Design UI

---

## 🖼️ Application Workflow

```text
Select Job Role
        ↓
Select Difficulty
        ↓
Generate Questions
        ↓
Technical Questions
        ↓
HR Questions
```

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/niftyrose/InterviewPrepAI.git
```

---

### Backend Setup

```bash
cd backend
pip install fastapi uvicorn
python -m uvicorn main:app --reload
```

Backend runs on:

```text
http://127.0.0.1:8000
```

---

### Frontend Setup

```bash
cd frontend
flutter pub get
flutter run -d chrome
```
